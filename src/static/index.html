Como Configurar

Guia passo a passo para conectar o Ultimate Compression com suas ferramentas de IA favoritas.

💻 1. Instalar em Nova Máquina / Servidor (Sem clonar Git)

Para instalar o Ultimate Compression globalmente na sua máquina, execute o comando correspondente no terminal:

No Linux / macOS (Bash):

curl -fsSL https://raw.githubusercontent.com/LucasGazula/ultimate-compression/main/install.sh | bash

No Windows (PowerShell):

irm https://raw.githubusercontent.com/LucasGazula/ultimate-compression/main/install.ps1 | iex

Nota: Após a instalação, basta executar uc start para subir o servidor e depois uc init dentro dos seus repositórios para ativar a compressão automaticamente!

🚀 2. Inicializar um Repositório (Uso diário)

Não é mais necessário editar arquivos manualmente ou configurar o eval a cada sessão. Apenas execute o comando a seguir no terminal do seu projeto:

python3 uc init

O que o comando faz automaticamente por você:

  • Configura as regras de comportamento do agente no projeto em .agents/AGENTS.md.
  • Registra o servidor MCP do Ultimate Compression globalmente no Antigravity CLI e Claude Code.
  • Injeta a configuração das variáveis de ambiente de proxy nos perfis de shell (.bashrc/.zshrc/.profile no Linux/macOS ou perfis do PowerShell no Windows).

🔄 3. Carregando no Terminal Atual

Para que o proxy do terminal atual reconheça a instalação do proxy imediatamente, recarregue seu terminal ou execute o comando de ativação manual uma única vez:

No Linux / macOS:

eval $(uc env)

No Windows (PowerShell):

uc env | iex

Dica: Todos os novos terminais abertos a partir de agora já iniciarão com o proxy ativo por padrão!
