Como Configurar

Guia passo a passo para conectar o Ultimate Compression com suas ferramentas de IA favoritas.

🤖 1. No Antigravity CLI / Claude Code

Modo Seguro (Nativo via MCP):

Abra o arquivo de configuração de MCP global (`~/.gemini/config/mcp.json` ou `~/.clauderc.json`) e insira a seguinte configuração:

{
  "mcpServers": {
    "ultimate-compression": {
      "command": "python3",
      "args": ["/mnt/ssd/ultimate-compression/src/mcp_server.py"]
    }
  }
}

💻 2. No VS Code / Cloud Code (Extensão)

Modo Proxy:

Se o seu assistente de IA usa chaves de API próprias, você pode configurar a variável de ambiente de API base local na sua sessão do terminal antes de abrir a IDE:

export OPENAI_API_BASE="http://localhost:20129/v1"
export ANTHROPIC_API_BASE="http://localhost:20129/v1"
code .

⚡ 3. Ativando o RTK (Shims Globais)

Para que ferramentas CLI interceptem automaticamente comandos verbose de Git e terminal, inicialize o ambiente na sua sessão atual:

eval $(uc env)

Ou adicione definitivamente no seu profile (`~/.bashrc` ou `~/.zshrc`):

export PATH="/mnt/ssd/ultimate-compression/shims:$PATH"

Créditos aos Criadores Originais

O Ultimate Compression é fruto do excelente trabalho dos seguintes desenvolvedores do ecossistema open-source:

🛠️ RTK (Rust Token Killer)

Autores: rtk-ai/rtk

Originalmente construído em Rust para compactar as saídas de ferramentas que os agentes usam no terminal.

🧠 Headroom

Autor: Tejas Chopra

Criador da filosofia de compressão reversível de contexto e gerenciamento inteligente de payloads.

🪨 Caveman Mode

Autor: Julius Brussee

Promotor do estilo conciso que descarta o excesso de formalismo em respostas de IA.

💈 Ponytail (Lazy Senior Dev)

Autor: Dietrich Gebert

Pioneiro nas regras de prompt que instruem o LLM a construir o menor código possível.

Agradecimentos especiais a todos os autores originais por disponibilizarem esses projetos para a comunidade! ❤️
